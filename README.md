# cgpt

text generation cli

## Installation

You can download and extract files for your own platform from [GitHub Releases](https://github.com/RyoMasumura1201/cgpt/releases)

## Usage

Set Your API KEY

```
export OPENAI_API_KEY="<YOUR_API_KEY>"
```

### Example

```
# Chat with GPT
cgpt "hello"

# Chat with GPT by VSCode
cgpt -e

# Reset chat
cgpt reset

# Switch GPT model
cgpt model

# Output config
cgpt show config

# Output chat
cgpt show chat

# Help
cgpt h
```

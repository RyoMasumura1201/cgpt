# cgpt

text generation cli

## Usage

Set Your API KEY

```
export OPENAI_API_KEY="<YOUR_API_KEY>"
```

### Example

```
# Chat with GPT
cgpt "hello"

# Chat with GPT by VSCode
cgpt -e

# Reset session
cgpt reset

# Switch GPT model
cgpt model

# Output config
cgpt show config

# Output session
cgpt show session

# Help
cgpt h
```
